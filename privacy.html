プライバシーポリシー（Privacy Policy）

VoiceMemo Transcriber / 最終更新日: 2026-02-19

binWord Inc.（以下「当社」）は、macOSアプリ「VoiceMemo Transcriber」における ユーザー情報の取扱いについて、以下のとおり定めます。

1. 取得・処理する情報

  • ユーザーが選択した監視フォルダ内の音声ファイル（例: m4a / wav / aiff / caf）
  • 音声ファイルの文字起こし結果（テキスト）
  • 処理履歴（成功/失敗、時刻、ファイル識別情報）
  • アプリ動作ログ（エラー情報を含む）

2. 利用目的

  • 音声ファイルの文字起こしを実行するため
  • 文字起こし結果をユーザーのNotesアプリに新規ノートとして保存するため
  • 処理結果の表示、障害解析、品質向上のため

3. 外部送信・第三者提供

本アプリは、当社サーバーへのユーザーデータ送信を目的として設計していません。 ただし、Appleが提供するOS標準機能（Speech / Notes / iCloud 等）の利用に伴い、 Appleの仕様に基づくデータ処理が行われる場合があります。 当社は、法令に基づく場合を除き、ユーザー情報を第三者へ提供しません。

4. データ保存場所

本アプリが扱うデータは、主にユーザーのMac上に保存されます。例:

  • ログ: ~/Library/Logs/VoiceMemoTranscriber/app.log（環境によりコンテナ配下）
  • 処理履歴DB: ~/Library/Application Support/VoiceMemoTranscriber/processed.sqlite3
  • 文字起こし結果ノート: ユーザーのNotesアプリ内

5. 権限

本アプリは、機能提供のために以下の権限を要求する場合があります。

  • Speech Recognition（文字起こし）
  • Automation / Apple Events（Notesへのノート作成）
  • 必要に応じてファイルアクセス関連権限

6. ユーザーによる管理

ユーザーは、macOSの設定から権限の許可/拒否を変更できます。 また、監視フォルダの変更、処理履歴の削除等を行うことができます。

7. 改定

本ポリシーは、必要に応じて改定されることがあります。 重要な変更がある場合は、公開ページ上で更新日を明示します。

Privacy Policy

VoiceMemo Transcriber / Last updated: 2026-02-19

binWord Inc. ("we", "us", or "our") describes how user information is handled in the macOS app "VoiceMemo Transcriber".

1. Information Processed

  • Audio files in the user-selected watch folder (e.g., m4a / wav / aiff / caf)
  • Transcription results generated from audio files
  • Processing history (success/failure, timestamps, file identifiers)
  • Application logs (including error information)

2. Purpose of Use

  • To transcribe audio files
  • To create new notes in the user's Notes app with transcription results
  • To display processing status, troubleshoot issues, and improve quality

3. Data Sharing

This app is not designed to send user data to our own servers. However, when using Apple platform features (Speech, Notes, iCloud, etc.), data may be processed according to Apple's platform behavior. We do not provide user information to third parties except as required by law.

4. Data Storage

Data is primarily stored on the user's Mac. Examples:

  • Logs: ~/Library/Logs/VoiceMemoTranscriber/app.log (or containerized location)
  • Processing history DB: ~/Library/Application Support/VoiceMemoTranscriber/processed.sqlite3
  • Transcriptions: saved to the user's Notes app

5. Permissions

The app may request the following permissions to provide core features:

  • Speech Recognition
  • Automation / Apple Events (to create notes in Notes)
  • File access related permissions when needed

6. User Controls

Users can manage permission grants in macOS Settings. Users can also change the watch folder and clear processing history.

7. Policy Updates

This policy may be updated when necessary. The "Last updated" date will reflect significant changes.
